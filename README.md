# 🚀 Project Overview

The **Pizza Sales Dashboard** is an interactive Power BI project designed to analyze pizza sales performance, customer purchasing behavior, product trends, and revenue insights.

This dashboard enables stakeholders to make data-driven decisions through powerful visualizations and KPI tracking.

# 📈 Key Insights

### Revenue Insights
- Generated total revenue of **488.53K**
- Average order value is **38.25**

### Category Insights
- Classic Pizza Category contributes the highest quantity sold.
- Revenue distribution is balanced across categories.

### Size Insights
- Large (L) pizzas generate the highest revenue.
- XL and XXL pizzas contribute the least revenue.

### Order Trends
- Peak order hours occur during afternoon and evening periods.
- Friday records the highest order volume.
